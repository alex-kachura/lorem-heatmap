# Lorem Heatmap

A heatmap generator. Based on [react-heatmap](https://github.com/JonathanWi/react-heatmap) by Jonathan Widawski

## Demo

https://lorem-heatmap.herokuapp.com/

## Run locally

```
yarn
yarn start
```

or

```
npm i
npm start
```
